# A PONG?
PONG suitable for 16-64 bit systems, tested using qemu and DOSBox, with slight flicker in DOSbos  
It also have boot sector!  
You can insert the floopy.img into your computer, turn on and play it.
# All the release follows in
floopy.bin: raw data of floopy disk  
floopy.img: floopy disk itself  
pong.com: a program work on the DOS and what you will see in the floopy
